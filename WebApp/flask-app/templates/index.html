The Project

Election Buzz powered by Big Neuron is an interactive web application that
allows the user to visualize big data pertaining to the 2016 Presidential Election, which is collected from Twitter and analyzed in our system. The data visualized in our application is based on the user's selected candidate.

Requirements for our project are as follows:

Data Processing

  • Tweepy API to pull Twitter data (realtime and batch processing)
  • Cassandra to ensure data persistence
  • Spark Streaming to generate realtime data
